interpolate griddata

Interpolate griddata using a Modeler instance that wraps all supported models. The model is selected with the argument model_name.

If the model_params is passed, then the model is initialized with those parameters. Otherwise, to make a search for the best parameters, use the model_params_grid argument.

The 3D grid for interpolation is retrived from the training data. At the moment features only a regular grid.

If requested, the griddata is preprocessed using the Preprocessor class.

Parameters

  • griddata (GridData) ‚Äì griddata to interpolate

  • model_name (str) ‚Äì model name

  • grid_resolution (float) ‚Äì grid resolution

  • model_params (dict, optional) ‚Äì model parameters. Defaults to {}.

  • preprocess_kwags (dict, optional) ‚Äì preprocessing parameters. Defaults to {}.

  • predict_kwags (dict, optional) ‚Äì prediction parameters. Defaults to {}.

  • return_model (bool, optional) ‚Äì return model. Defaults to False.

Returns

interpolated griddata, optionally with model

Return type

Union[np.ndarray, Tuple[np.ndarray, Modeler]]

Raises

  • ValueError ‚Äì either model_params or model_params_grid must be passed

  • ValueError ‚Äì model_params and model_params_grid cannot be passed together

  • NotImplementedError ‚Äì only RegularGrid3D is supported.

  • NotImplementedError ‚Äì Parameter search is only supported for ordinary_kriging
