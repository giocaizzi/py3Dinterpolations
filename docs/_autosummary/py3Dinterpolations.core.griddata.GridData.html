class py3Dinterpolations.core.griddata.GridData(data: DataFrame, ID: str = 'ID', X: str = 'X', Y: str = 'Y', Z: str = 'Z', V: str = 'V', preprocessing_params: dict = {})

Bases: object

GridData class

Class for 3D grid data.

As defaults, looks for X, Y, Z, V columns in the data. Otherwise, custom column names can be provided.

This object keeps track of the transformations (preprocessing) applied to the data, by storing the relative parameters in the preprocessing_params attribute.

Parameters

  • data (pd.DataFrame) – source data

  • ID (str) – column name of the ID, default: “ID”

  • X (str) – column name of the X, default: “X”

  • Y (str) – column name of the Y, default: “Y”

  • Z (str) – column name of the Z, default: “Z”

  • V (str) – column name of the V, default: “V”

  • preprocessing_params (dict) – preprocessing parameters
